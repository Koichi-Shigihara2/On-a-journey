METHODOLOGY

V0 本質的価値

過去5年平均FCFを起点に、2段階DCFで算出。 高成長期（3年）はFCF CAGRを適用し、その後ターミナル成長率3%で永続価値を算出。
WACC = 8.5%

α 成長期待プレミアム

過去10年平均ROE × 内部留保率から個別成長率を算出。
α = (g個別 / WACC) × 0.7
確度調整により過大評価を抑制。

FCF補正

FCFがマイナスの場合、売上高×8%をフロアとして適用。 成長投資期の企業でも合理的な本質価値を算出。

1〜3年後予測

高成長期間中は企業別CAGRを適用、以降はターミナル成長率。 Phase 4では将来価値予測を自動生成。
