KOICHI式 v5.1 計算モデル

Pt = V0 × (1 + α)

本質的価値 = DCF企業価値 × (1 + 成長期待プレミアム)

V0 = Σ[FCF×(1+g)t / (1+WACC)t] + TV/(1+WACC)n

2段階DCF: 高成長期(3年) + ターミナル価値。WACC=8.5%、ターミナル成長率=3%

α = max(0, (ROE × 内部留保率 / WACC) × 0.7)

成長期待プレミアム: 過去10年平均ROEと内部留保率(60%)から算出
